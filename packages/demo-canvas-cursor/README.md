# @mlightcad/demo-canvas-cursor

Demo app for `@mlightcad/text-box-cursor` using the `Canvas2DRendererAdapter`.

## Purpose

- Show cursor rendering, selection rendering, and debug overlays on HTML canvas.
- Validate pointer interactions and zoom behavior in a 2D renderer.
- Verify keyboard navigation and selection extension behavior with the same cursor core API.

## Run

```bash
pnpm --filter @mlightcad/demo-canvas-cursor dev
```

## Build

```bash
pnpm --filter @mlightcad/demo-canvas-cursor build
```
