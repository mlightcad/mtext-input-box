# @mlightcad/demo-three-cursor

Demo app for `@mlightcad/text-box-cursor` using the `ThreeJsRendererAdapter`.

## Purpose

- Show cursor and selection rendering in a Three.js scene.
- Validate text draw calls, debug overlays, and camera zoom/pan behavior.
- Verify pointer-based cursor placement and drag selection interactions.

## Run

```bash
pnpm --filter @mlightcad/demo-three-cursor dev
```

## Build

```bash
pnpm --filter @mlightcad/demo-three-cursor build
```
